 Верстка

При верстке следующих работ я старался применять новые технологии. Поэтому в IE11 они будут иметь некоторые визуальные отличия. Если есть такая необходимость, я могу верстать с поддержкой IE9+, используя bootstrap 3, susy или другие библиотеки.

1 ) При верстке всех работ использовал Gulp, SCSS, сброс стилей, БЭМ (только именование классво), иконочный шрифт Font Awesome (fontello), Flexbox, слайдер tiny-slider, адаптивные изображения (picture, srcset), humburger menu на узких экранах, box-shadow, transition
2 ) При билде для:
a ) JS: uglify, concat, babel.
b ) CSS: groupmedia, cleanCSS
c ) картинок: imagemin, webp

REACT PIZZA

REACT PIZZA - это онлайн магазин.

Клиентская часть написана с помощью React, управление сотоянием - Redux, лоудер при загрузке react-content-loader, маршрутизация - react-router-dom.

Серверная часть написана на Node.js с помощью библиотеки Express. Данные хранятся в БД mongoDB. Подключение к БД осуществляется с помощью библиотеки mongoose

К сожалению, это не законченная работа. Поэтому перечислю что сделано и что в работе.
Сделана сортировка по категориям, фильтр по категориям. Выбор вида и размера теста пиццы, работа корзины.
В планах сделать админку, логин для администратора, редактирование пицц, категорий сортировки и фильтра, подключение платежей.
